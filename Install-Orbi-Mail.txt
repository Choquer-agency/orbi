Orbi Mail — Install Instructions
=================================

You got two files:
  • Orbi Mail-arm64.dmg  → for Apple Silicon Macs (M1 / M2 / M3 / M4)
  • Orbi Mail-x64.dmg    → for Intel Macs (older, pre-2020)

Not sure which one you have?
  Apple menu  →  About This Mac  →  look at the "Chip" or "Processor" line.
    "Apple M…"  → use the arm64 file
    "Intel …"   → use the x64 file


How to install
--------------
1. Double-click the DMG that matches your Mac.
2. Drag "Orbi Mail" into the Applications folder.
3. Eject the DMG.


First launch (important — it's unsigned)
----------------------------------------
The first time you open it, macOS will say it can't verify the developer.

Do this:
  1. Open Applications, RIGHT-CLICK (or Control-click) "Orbi Mail" → Open.
  2. Click "Open" in the dialog.

If it still won't launch:
  • System Settings → Privacy & Security
  • Scroll down to the Security section
  • You'll see a message about "Orbi Mail" being blocked → click "Open Anyway"
  • Confirm with your password / Touch ID.

After that first launch, it'll open normally from then on.


Signing in
----------
On the login screen, click "Sign up" to create your account.
Then connect your Gmail or Outlook mailbox from Settings → Accounts.
